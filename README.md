## 🚀 我们的使命

Made-in-AGI 致力于连接全球AI供需双方，为企业提供一站式AI采购与交易服务，覆盖大模型、智能体、算力和行业解决方案。

## 🔗 核心业务

- **🤖 AGI 大模型服务**：提供通用及垂直领域大模型的私有化部署、API接口和模型微调。
- **🧠 AI 智能体（Agent）**：覆盖客服、营销、办公自动化等多场景的AI智能体。
- **💻 硅基算力**：GPU高性能算力租赁、液冷算力中心方案等基础设施服务。
- **⚙️ 定制化AI开发**：面向政企客户的全栈AI定制开发和系统集成服务。
